01
Research Paper · CMSC 190 Special Problem

Classification of High and Low Protein Corn Using NIR Spectral Data and Machine Learning

View Full Preview

Benchmarked three ML models — PLS-DA, SVM-RBF, and a 1D-CNN — on NIR spectral data from 549 maize samples for binary protein classification. Applied Savitzky-Golay smoothing and stratified 80/20 splitting. Deployed the top model (PLS-DA) into a prototype web application with a Next.js frontend and FastAPI backend, allowing users to upload CSV spectra and receive real-time protein class predictions.
